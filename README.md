# PCStore
questa app è un servizio di e-commerce che è stato realizzato durante il corso di sviluppatore web back-end
le tecnologie utilizzate per realizzare questo progetto sono: 
-ThymLeaf
-SpringDataJDBC
-MySQL
-SpringWeb
